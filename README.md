📚 Programming Learning App

A Flutter-based educational mobile application designed to help beginners learn programming fundamentals through structured lessons, video tutorials, and interactive quizzes.

🚀 Overview

This application delivers a simple, modern, and effective learning experience for beginners in programming. It combines structured content, video lessons, and quizzes in a clean UI to make learning more engaging and practical.

The project also demonstrates clean architecture principles and scalable Flutter development practices.

✨ Features
📖 Structured programming lessons (C++, Dart, and more)
🎥 Video-based learning modules
🧠 Interactive quizzes after each lesson
🔐 User authentication (login & registration using Firebase)
🎨 Clean and modern UI design
📱 Fully responsive layout using flutter_screenutil
♻️ Reusable and scalable widget architecture
🧰 Tech Stack
Flutter (Dart)
Firebase Authentication
Material Design
flutter_screenutil
Custom reusable components
📁 Project Structure
lib/
├── core/        # App core (themes, constants, utilities)
├── features/    # Main features (auth, home, courses)
├── widgets/     # Reusable UI components
└── main.dart    # App entry point
🎯 Project Goals

This project was built to:

Help beginners understand programming fundamentals
Provide an interactive and structured learning experience

⚙️ Getting Started

Follow these steps to run the project locally:

1️⃣ Clone the repository
git clone https://github.com/your-username/programming-app.git
2️⃣ Install dependencies
flutter pub get
3️⃣ Run the app
flutter run
🔐 Notes
Make sure Firebase is properly configured before running the project.
Store sensitive data (API keys, environment variables) in a .env file and exclude it from version control.
Do not upload Firebase config files publicly unless properly secured.
